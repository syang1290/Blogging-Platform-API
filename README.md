Blogging Platform API Project

Built a RESTful API that contains basic methods such as create, update, delete, get, get all in Python
Covers all the CRUD Operations

The data is stored locally in a SQLite database.
Routing on the navbar allows users to create a post then see the list of their created posts

Skills Used:

Backend - Python, Flask, SQLite3
Frontend - React/Vite, HTML, Javascript, CSS
